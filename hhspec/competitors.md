## 1. birdygrey.com

- **URL**: https://birdygrey.com
- **定位**: 伴娘礼服垂直电商，主打"colors and styles that make everyone happy"
- **价格**: $89–$199（多数 $129）
- **核心技术特征**:
  - **色板系统（Swatches）**: 数十种颜色，Shop by Color 是首页一级入口，色块网格导流
  - **Try-at-Home**: 邮寄色板试色再下单，解决"选色焦虑"
  - **Free Moodboards**: 色彩规划工具，导流色板购买
  - **Convertible 可变穿法**: 一款多穿 + How-to-Wear 引导
  - **交期 SKU 化**: Standard / Rush Production 做成可选维度
  - **颜色族组织全站**: Color Families 是 PLP 首要 Filter
- **调性**: 明亮淡雅、大留白、浅色背景产品图 + 户外 lifestyle 混排
- **素材量**: 16 张产品图（伴娘裙多色 + 配饰 + 花童）

## 2. davidsbridal.com

- **URL**: https://davidsbridal.com
- **定位**: 综合婚礼电商，O2O（线上 + 实体店预约 + Fit Guarantee）
- **价格**: 伴娘 $24.94 起，婚纱 $99–$500+
- **核心技术特征**:
  - **全角色覆盖**: Brides / Bridesmaids / MOB / Guest / Flower Girl / Prom
  - **Fit Guarantee**: 合身保障 + Alterations + Book Appointment + Find a Store
  - **Extra Length 定制**: 裙长定制做成 SKU 维度
  - **感性色名体系**: Dusty Sage / Steel Blue / Desert Coral / Dusty Blue / Ballet / Petal / Martini Olive / Lavender Haze
  - **最完整 Filter**: Color / Size / Price / Brand / Collection / Fit / Silhouette / Neckline / Sleeve / Waist / Train / Length / Fabric / Occasion / Season
  - **尺码体系**: 0–24，含 Plus / Petite
- **内容**: Inspiration / Real Weddings / Registry / Diamond Loyalty
- **素材量**: 18 张图（婚纱 + 伴娘裙多色 + 3 张运营 banner）

## 3. kissprom.com

- **URL**: https://kissprom.com
- **定位**: Prom / 晚礼服 + 婚纱跨场景跨境电商
- **价格**: Prom $150–$300，Wedding $259+，Homecoming Under $100
- **核心技术特征**:
  - **最细硬 Filter**: Neckline / Sleeve / Length / Back Style / Back Type / Fabric / 48h 现货
  - **Custom Size 量身定制**: US 00–30 + 定制尺寸（核心差异化）
  - **描述化商品命名**: 版型 + 色 + 面料 + 领型 + 裙长全写进标题（如 "A-Line One Shoulder Sage Green Long Prom Dress with Appliques"）
  - **场景高度细分**: Prom / Homecoming / Wedding / Bridesmaid / Evening / Cocktail / Quinceañera / Graduation
  - **Influencer Program**: 网红合作 + Prom Guide 内容营销
- **价值主张**: "thousands of gowns in every style and size"、"fast USA shipping and easy returns"
- **素材量**: 18 张图（Prom 多色 + 婚纱 6 款 + Homecoming 3 款）

## 4. azazie.com

- **URL**: https://www.azazie.com
- **定位**: 伴娘 + 婚纱垂直 DTC，按需定制（made-to-order），主打"试穿前置 + 免费定制尺寸"，与 Birdy Grey 同赛道但产品力更重
- **价格**: 伴娘裙 $69 起；600+ 款式、90+ 颜色
- **核心技术特征**:
  - **免费色板（Free Swatch Program）**: 注册可领 10 个免费色板；面料含 Chiffon / Stretch Satin / Metallic Satin / Floral Burnout
  - **At-Home Try-On（标杆差异化）**: 任选 3 件样衣（伴娘/Junior/Moms/婚纱），平台免费寄送 + 附退货标签，1 周试穿；移动端有 "Try at Home" 筛选
  - **免费定制尺寸（Custom Sizing）**: 全线 0–30 码 + 提供精确量体尺寸定制，**不额外收费**；所有裙子按单手工裁剪缝制（made-to-order）
  - **Showroom 协作规划**: 收藏款式 + 分享/投票，便于伴娘团协同决策（社交化选款）
  - **配饰联动**: 领带 / 手帕 / 披肩 / 腰带 / 围巾，按婚礼色系匹配
- **品类导航**: Bridesmaids / Brides / Moms / Flower Girl / Formal & Cocktail / Prom & Party / Shoes & Accessories / Suits & Ties / Intimate / Home Try-On；子类 Boho / Maternity / Modest / Beach / Mix-and-Match / Junior Bridesmaid
- **退货**: 标准码 10 天可退；定制码不可退
- **借鉴价值**: 「免费定制尺寸 + 免费 At-Home Try-On + Showroom 协作」是高端户外婚礼"为整组伴娘配色配码"的最佳范本；调性同样明亮淡雅，与本项目高度契合
- **素材量**: 未下载（仅结构分析）

## 5. jjshouse.com

- **URL**: https://www.jjshouse.com
- **定位**: 全球综合礼服跨境电商（2010 成立，中国发货），2000+ 款式，覆盖 regular / petite / plus
- **价格**: 性价比定位，多数客户反馈"$2000 婚纱级质感、平价价格"
- **核心技术特征**:
  - **Color & Fabric Swatch**: 色板/面料寄样（小额收费）
  - **Try-Before-You-Buy**: $10 试穿服务，含寄送与退货
  - **Custom Size**: 按量体定制（定制款基本不可退）
  - **Global Shipping**: 全球物流（时效较长，是主要痛点）
  - **奖项背书**: The Knot Best of Weddings 2021/2024/2025、WeddingWire Couples' Choice 2022–2025、Google Trusted Store
- **品类导航**: Wedding / Bridesmaid / Mother of Bride & Groom / Prom & Homecoming / Accessories（含 Veils 头纱）/ Wedding Suits / Shawls / Decor
- **已知痛点（反面借鉴）**: 尺码不一致、定制款难退、物流/退款慢、运费不退、客服工单藏在 "Tickets" 区不易发现 → 原型应在尺码表/退换政策/物流时效/工单可见性上做正向设计
- **借鉴价值**: 海量 SKU 的跨境运营骨架 + Try-Before-You-Buy + 奖项/Trusted Store 信任背书；其痛点反向印证本项目应重点打磨"尺码准确性 + 透明物流 + 清晰退换"
- **素材量**: 未下载（仅结构分析）

## 五家共性

- **平台**: birdygrey / davidsbridal / kissprom 为 Shopify；azazie / jjshouse 为自建 DTC/跨境系统
- **导航三维交叉**: 场景/角色 × 颜色 × 版型/面料
- **颜色是第一公民**: 均把"按颜色购物"做成首页入口 + PLP 首要 Filter
- **试穿/试色前置**: 色板（5/5 家）+ At-Home/Try-Before-You-Buy 实物试穿（birdygrey / azazie / jjshouse）已成行业标配钩子
- **定制尺寸**: kissprom / azazie / jjshouse 均提供 Custom Size，azazie 免费且为核心卖点
- **PDP 标配**: 图廊 + Color/Size + 尺码表 + Description + Fabric + Shipping/Returns + Reviews + 关联推荐
- **评价权重高**: 均为转化要素
- **页脚一致**: About / Contact / FAQ / Size Guide / Shipping / Returns / Privacy / Terms + Newsletter + 社交

**关键发现**
- **Shopify 新版账户已 passwordless**：birdygrey / kissprom 均无密码字段，输入邮箱后收 6 位验证码登录（Shopify Customer Accounts 默认形态），降低注册摩擦。
- **Google 最普遍**：3 家已确认全部支持「Continue with Google」。
- **davidsbridal 用自建 OIDC**：保留传统「邮箱+密码」，并叠加 Google + Apple；其 `/account/register` 直接 404，注册入口收敛到 login 页的 "Sign up"。
- **Facebook**：kissprom 实地确认；jjshouse 据其官方帮助中心亦支持（Google/Facebook 社交登录）。
- **Apple**：仅 davidsbridal 实地确认。
- 各家均未见手机号/短信注册（面向欧美市场，以邮箱为主）。
- **azazie 未能确认**：Cloudflare challenge 全程拦截 headless/headed 抓取，WebSearch 亦无官方记载；如需确认建议人工登录其 `/account` 页核对。

**对 Dreamy 原型的建议**
- 主路径采用「邮箱 + 验证码（passwordless）」+「Continue with Google」，对齐行业当前主流、最低摩擦。
- Apple 登录作为可选增强（iOS 用户友好，davidsbridal 已采用）。
- 不做手机号注册（目标市场为高端欧美户外婚礼）。
