# UI 验收清单 — fabric-care-module

> 变更名称：fabric-care-module
> 生成时间：2026-06-14T08:20:00Z
> 门户：portal-admin
> 基于：operation-paths.yml + content-index.json

## 验收目标

验证面料护理模块的两个核心编辑器组件在商品编辑页（ProductEdit.vue）的 UI 交互正确性。

## 测试环境

- **测试页面**：http://localhost:5174/products/edit/:id
- **测试浏览器**：Chrome 最新版、Safari 最新版
- **测试分辨率**：1920x1080、1366x768
- **测试数据**：至少 1 个已存在的商品（ID=1）

## UI-001 面料成分编辑器（FabricCompositionEditor）

### UI-001-01 组件渲染

**Given** 打开商品编辑页  
**When** 滚动到"面料成分"区域  
**Then** 应显示：
- [ ] 区域标题："面料成分 / Fabric Composition"
- [ ] Tab 切换栏：Shell（外层）、Lining（内衬）、Overlay（叠层）、Trim（装饰）
- [ ] 默认选中 Shell Tab
- [ ] Shell Tab 下方显示成分表格（或"添加材料"按钮）

### UI-001-02 添加材料

**Given** Shell Tab 已激活且无成分数据  
**When** 点击"添加材料"按钮  
**Then** 应显示：
- [ ] 新增一行表格
- [ ] 第一列为材料下拉选择器（含 20 种材料选项，按分类分组）
- [ ] 第二列为百分比输入框（placeholder: "0-100"）
- [ ] 第三列为删除按钮（X 图标或 "删除" 文字）

### UI-001-03 材料选择

**Given** 已添加一行成分  
**When** 点击材料下拉选择器  
**Then** 应显示：
- [ ] 5 个分组选项（天然纤维、合成纤维、半合成纤维、弹性纤维、特殊面料）
- [ ] 每个分组下列出对应材料，格式："中文名 / English"
- [ ] 选项可滚动查看

**When** 选择 "涤纶 / Polyester"  
**Then** 应：
- [ ] 下拉关闭
- [ ] 材料列显示 "涤纶 / Polyester"
- [ ] 焦点自动移动到百分比输入框

### UI-001-04 百分比输入与校验

**Given** 已选择材料  
**When** 在百分比输入框输入 "80"  
**Then** 应：
- [ ] 接受输入
- [ ] 底部显示当前层总百分比："总计：80%"（黄色或橙色警告）

**When** 点击"添加材料"再添加一行，材料选择 "氨纶 / Elastane"，百分比输入 "20"  
**Then** 应：
- [ ] 底部显示："总计：100%"（绿色对勾或正常颜色）

**When** 修改第一行百分比为 "90"  
**Then** 应：
- [ ] 底部显示："总计：110%"（红色错误）
- [ ] 显示错误提示："百分比总和必须等于 100%"

**When** 尝试输入非数字字符（如 "abc"）  
**Then** 应：
- [ ] 拒绝输入或自动过滤非数字字符
- [ ] 仅接受 0-9 和小数点

**When** 尝试输入负数（如 "-10"）  
**Then** 应：
- [ ] 拒绝输入或显示错误提示："百分比必须在 0-100 之间"

### UI-001-05 删除材料

**Given** 已添加 2 行成分  
**When** 点击第一行的删除按钮  
**Then** 应：
- [ ] 该行立即移除
- [ ] 重新计算总百分比并更新显示
- [ ] 剩余行不受影响

### UI-001-06 切换层级 Tab

**Given** Shell Tab 已填写 2 行成分（总计 100%）  
**When** 点击 Lining（内衬）Tab  
**Then** 应：
- [ ] Tab 切换到 Lining
- [ ] Shell Tab 显示勾选标记或数字角标（表示已完成）
- [ ] Lining Tab 内容区域为空或显示"添加材料"按钮
- [ ] Shell 的数据保持不变（切换回 Shell 应能看到之前的数据）

### UI-001-07 保存状态

**Given** 至少填写了 Shell 层成分（总计 100%）  
**When** 点击页面底部的"保存"或"提交"按钮  
**Then** 应：
- [ ] 触发表单提交
- [ ] 显示加载状态（按钮禁用 + 加载图标）
- [ ] 成功后显示 toast 提示："保存成功"
- [ ] 失败时显示 toast 提示："保存失败：{错误原因}"

## UI-002 护理标签选择器（CareSymbolSelector）

### UI-002-01 组件渲染

**Given** 打开商品编辑页  
**When** 滚动到"护理说明"区域  
**Then** 应显示：
- [ ] 区域标题："护理说明 / Care Instructions"
- [ ] 5 个分类手风琴（Washing、Bleaching、Drying、Ironing、Professional Care）
- [ ] 默认第一个分类（Washing）展开
- [ ] 底部显示"已选符号预览"区域（初始为空或显示"尚未选择护理符号"）

### UI-002-02 展开/折叠分类

**Given** Washing 分类已展开  
**When** 点击 Washing 分类标题  
**Then** 应：
- [ ] Washing 分类折叠（内容隐藏）
- [ ] 箭头图标旋转（向下 → 向右）

**When** 点击 Bleaching 分类标题  
**Then** 应：
- [ ] Bleaching 分类展开
- [ ] 显示该分类下的所有符号卡片（3 个）
- [ ] 其他分类状态不变（允许多个分类同时展开）

### UI-002-03 符号卡片展示

**Given** Washing 分类已展开  
**Then** 应显示：
- [ ] 8 个符号卡片，Grid 网格布局（每行 4-5 个）
- [ ] 每个卡片包含：
  - [ ] 符号图标（40x40px，SVG 或图片）
  - [ ] 简短说明文字（中文或中英双语）
  - [ ] 复选框（位于卡片角落或边缘）
- [ ] 卡片有边框和圆角
- [ ] Hover 时卡片有阴影或高亮效果

### UI-002-04 选择符号

**Given** Washing 分类已展开  
**When** 点击 "Machine wash cold (30°C)" 符号卡片  
**Then** 应：
- [ ] 卡片边框变为高亮颜色（如蓝色）
- [ ] 复选框勾选
- [ ] 底部"已选符号预览"区域增加该符号的图标
- [ ] 预览区图标横向排列，间距均匀

**When** 再次点击同一符号卡片  
**Then** 应：
- [ ] 取消选中状态
- [ ] 卡片边框恢复普通颜色
- [ ] 复选框取消勾选
- [ ] 底部预览区移除该符号图标

### UI-002-05 多选符号

**Given** 已选择 Washing 类别的 "Machine wash cold (30°C)"  
**When** 展开 Bleaching 分类，选择 "Non-chlorine bleach only"  
**Then** 应：
- [ ] 两个符号都处于选中状态
- [ ] 底部预览区显示 2 个符号图标
- [ ] 预览区图标按选择顺序排列

**When** 继续在其他分类中选择符号（最多选择 10 个）  
**Then** 应：
- [ ] 所有选中的符号在预览区显示
- [ ] 预览区图标自动换行（如果空间不足）
- [ ] 预览区可滚动查看

### UI-002-06 符号预览交互

**Given** 底部预览区已有 3 个已选符号  
**When** 鼠标悬停在预览区某个符号图标上  
**Then** 应：
- [ ] 显示 tooltip 提示框
- [ ] Tooltip 包含符号的完整说明（中英文）

**When** 点击预览区的某个符号图标  
**Then** 应：
- [ ] 取消选中该符号
- [ ] 对应分类中的符号卡片取消勾选
- [ ] 预览区移除该符号图标

### UI-002-07 保存状态

**Given** 已选择至少 3 个护理符号  
**When** 点击页面底部的"保存"或"提交"按钮  
**Then** 应：
- [ ] 与面料成分一起提交
- [ ] 成功后护理符号保持选中状态
- [ ] 刷新页面后能恢复之前的选择

## UI-003 集成测试

### UI-003-01 两个组件同时使用

**Given** 同时填写了面料成分和护理标签  
**When** 保存商品  
**Then** 应：
- [ ] 两部分数据都正确提交到后端
- [ ] 保存成功后刷新页面，数据正确回显

### UI-003-02 表单验证

**Given** 面料成分 Shell 层总百分比不等于 100%  
**When** 点击保存按钮  
**Then** 应：
- [ ] 显示错误提示："面料成分百分比总和必须等于 100%"
- [ ] 自动滚动到面料成分区域
- [ ] 高亮错误的 Tab（Shell）
- [ ] 阻止表单提交

### UI-003-03 空数据处理

**Given** 未填写任何面料成分和护理标签  
**When** 保存商品  
**Then** 应：
- [ ] 允许保存（面料信息为可选项）
- [ ] 或显示提示："建议填写面料成分以提升商品信息完整度"（非阻断）

## UI-004 响应式与兼容性

### UI-004-01 窗口缩放

**Given** 打开商品编辑页  
**When** 将浏览器窗口缩小到 1366x768  
**Then** 应：
- [ ] 组件布局自适应，不出现水平滚动条
- [ ] 符号卡片网格列数减少（如 5 列 → 4 列）
- [ ] 所有按钮和输入框可点击

### UI-004-02 浏览器兼容性

**When** 在 Safari 浏览器中打开  
**Then** 应：
- [ ] 所有功能正常工作
- [ ] 样式与 Chrome 一致或仅有细微差异
- [ ] SVG 图标正常显示

## UI-005 性能与可用性

### UI-005-01 加载性能

**When** 首次渲染面料成分编辑器和护理标签选择器  
**Then** 应：
- [ ] 组件渲染时间 < 500ms
- [ ] 下拉选择器打开延迟 < 100ms
- [ ] Tab 切换无明显延迟

### UI-005-02 键盘导航

**Given** 焦点在材料下拉选择器  
**When** 按下 Tab 键  
**Then** 应：
- [ ] 焦点移动到百分比输入框

**When** 按下 Enter 键（焦点在复选框上）  
**Then** 应：
- [ ] 切换符号选中状态
